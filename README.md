# Birthday Board 🎂

A personal website to track birthdays of friends and family — styled with **Neo Brutalism** inspired by the [Neo Brutalism UI Library on Figma](https://www.figma.com/community/file/1613945708230609539/neo-brutalism-ui-library) (shadcn/ui neo-brutalism style).

Bold borders. Hard shadows. No softness allowed.

## Features

- **12-month picker** — chunky tab buttons with birthday counts
- **Coming up** — next 5 birthdays sorted by days until
- **Add, edit, delete** birthdays with relationship tags and notes
- **Local persistence** — saved in your browser via localStorage
- **Neo-brutalist UI** — 3px black borders, 4px offset shadows, saturated flat colors

## Quick start

```bash
cd birthday-dj
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).
